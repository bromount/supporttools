This is script is to backup all the articles in a zendesk site.

Give the zendesk sitename, username, password and email address to email the backup file in config.txt

example:

[settings]

site = https://orchestrate.zendesk.com
username = jeyanthan@collab.net
password = Collabnet1!
email_from = shrinivasan@collab.net
email_to = shrinivasan@collab.net

How to run?

Just run as below

python backup-zendesk-topics.py
